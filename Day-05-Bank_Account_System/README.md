# Day 05 - Bank Account System

## 📌 Overview

A simple Bank Account System built using Python Object-Oriented Programming (OOP).

This project simulates basic banking operations such as creating an account, depositing money, withdrawing money, and checking the account balance. It also demonstrates the concept of **Encapsulation** by keeping the account balance private.

---

## ✨ Features

* Create a bank account
* Deposit money
* Withdraw money
* Check current balance
* Prevent invalid withdrawals
* Demonstrates Encapsulation using private variables

---

## 🛠 Technologies Used

* Python

---

## 📚 Concepts Used

* Classes & Objects
* Constructor (`__init__`)
* Methods
* Encapsulation
* Private Variables (`__balance`)
* Conditional Statements (`if-else`)
* Object Creation
