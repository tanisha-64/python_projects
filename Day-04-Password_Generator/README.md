# Password Generator

## Overview
A simple, interactive command-line Python application that generates secure, random passwords based on user-defined lengths and character preferences (letters, numbers, and symbols).

## Features
- **Custom Length:** Choose how long you want your password to be.
- **Character Filtering:** Toggle the inclusion of letters, numbers, and special characters.
- **Input Validation:** Error handling to prevent system crashes on invalid inputs.

## Technologies Used
- **Python 3.x**
- Built-in modules: `random`, `string`

## How to Run
1. Clone this repository or download the folder.
2. Navigate into the project folder:

   cd Password-Generator
