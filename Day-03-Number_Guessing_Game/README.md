# Day 03 - Number Guessing Game

## 📌 Overview

A simple Python game where the computer randomly selects a number between **1 and 100**, and the player has to guess the correct number.

The program provides hints if the guess is too high or too low until the correct number is guessed.

---

## ✨ Features

* Random number generation
* User input
* Unlimited guessing attempts
* Hint system (Too High / Too Low)
* Attempt counter
* Beginner-friendly code

---

## 🛠 Technologies Used

* Python

---

## 📚 Concepts Used

* Variables
* Input / Output
* while Loop
* if-elif-else
* Comparison Operators
* Random Module
* Counter Variable
