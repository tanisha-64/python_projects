# Day 02 - Calculator

## Overview

A simple calculator built using Python that performs basic arithmetic operations.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* User-Friendly Menu

## Technologies Used

* Python

## Concepts Used

* Variables
* Input / Output
* Arithmetic Operators
* Conditional Statements (if-elif-else)
* Basic Error Handling

## How to Run

```bash
python calculator.py
```

## Sample Output

```text
=== SIMPLE CALCULATOR ===

Enter first number: 10
Enter second number: 5

Choose Operation
1. Addition
2. Subtraction
3. Multiplication
4. Division

Enter choice (1-4): 1

Result = 15
```

## Learning Outcomes

* Taking user input
* Performing calculations
* Using conditional statements
* Building a simple command-line application
